# op-shopper

App for op shops to list items online for customers to browse.

# Boilerplate: Knex.js

Phase 1 boilerplate project with:

 - Express
 - Knex
 - SQLite3
 - Express Handlebars
 - body-parser
 - Jest
 - nodemon


## Install

```
yarn
yarn knex migrate:latest
yarn knex seed:run
yarn dev
```

Be sure to check out the other npm scripts too.
